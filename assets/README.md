## Recommended Additions

- `electrical_architecture.png`: system-level electronics overview.
- `mechanism_animation.gif`: short morphing-wing mechanism animation.
- `experiment_setup.jpg`: pool-test or laboratory setup photo.
- `trajectory_example.png`: representative raw/processed trajectory visualization.

Keep source-editable figures in the corresponding subsystem directory when available, and place only GitHub-facing preview media here.
