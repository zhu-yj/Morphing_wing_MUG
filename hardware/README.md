# Hardware

This directory is reserved for mechanical design files and build documentation.

## Recommended Layout

| Directory | Contents |
| --- | --- |
| `cad/` | native CAD and neutral CAD exports |
| `drawings/` | manufacturing drawings, DXF/PDF drawings |
| `bom/` | mechanical and system bill of materials |
| `assembly/` | assembly sequence, inspection, sealing notes |
| `fabrication/` | manufacturing tolerances and fabrication notes |

## Release Checklist

- Add native CAD files if redistribution is allowed.
- Add STEP/STL exports for users without the native CAD software.
- Add critical dimensions and tolerance drawings.
- Add mass, center-of-mass, and buoyancy notes if available.
- Remove vendor-restricted or institution-only files before public release.
